# 老年人服务预约系统 👴📱

## 📋 软件简介
这是一款帮助老年人线上预约护工上门服务的软件，目前已完成基础功能开发。

### ✨ 主要功能
- **用户注册/登录**：用户和员工信息统一存入 `users_data.txt`
- **员工工作状态管理**：员工登录后可切换工作状态（可预约/不可预约）
- **用户预约功能**：用户可查看所有在职员工的姓名、年龄、电话、薪资要求，选择心仪员工
- **一对一预约**：员工被选择后，显示签约用户信息（姓名、电话、住址）
- **取消预约**：双方均可自由取消预约

## 🛠️ 开发环境
| 组件 | 版本 |
|------|------|
| IDE | IntelliJ IDEA Community Edition 2025.2.1 |
| 语言 | Java 24 |
| JDK | Zulu 24 |
| GUI框架 | JavaFX 25 |
| 数据存储 | 本地文件 (users_data.txt) |

## 🚀 快速开始

### 1️⃣ 环境配置
请确保已安装以上开发环境组件。

### 2️⃣ 运行程序
1. 用IDEA打开项目
2. 配置JavaFX 25
3. 运行 `Main.java`

### 3️⃣ 首次使用
- **新用户**：点击注册创建账号
- **已有账号**：直接登录
- **员工注意**：新账号登录后需点击红色“暂停工作”按钮恢复工作状态（才能被预约）
